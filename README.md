#注意事项
# 1.react的内容要放在body内 置于容器的下面
# 2.使用jsx 一定要将将script的type定义成 text/jsx并引入JSXTransformer.js


#demo

```
<body>

<!-- 定义一个容器承载内容 begin -->
<div id="content"></div>
<div id="moreChildren"></div>

<!-- 真正的代码要开始了  -->
<script src="index1.js"></script>

</body>
```
